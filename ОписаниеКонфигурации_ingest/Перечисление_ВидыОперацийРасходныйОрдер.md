# Перечисление: ВидыОперацийРасходныйОрдер (Виды операций документа "Расходный ордер")

## Значения

- ВозвратПоПриходномуОрдеру (возврат по приходному ордеру)
- РасходПоНакладной (расход по накладной)
- ВозвратПоставщику (возврат поставщику)
- Перемещение (перемещение)
- ВПереработку (передача материалов переработчику)
- ИзПереработки (передача продукции заказчику)
- НеОпределен (не определен)
