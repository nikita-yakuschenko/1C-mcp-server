# Перечисление: ВариантыОбработкиЗаписиПриОтраженииВРеглУчете (Варианты обработки записи при отражении в регл учете)

## Значения

- НеОбрабатывать (Не обрабатывать)
- БольничныйТравмаНаПроизводстве (Больничный травма на производстве)
- ОтпускПоБеременностиИРодамПоУходуЗаРебенком (Отпуск по беременности и родам, по уходу за ребенком)
- БольничныйПрочий (Больничный прочий)
- БольничныйОбщееЗаболевание (Больничный общее заболевание)
- БольничныйТравмаНаПроизводствеНовыйСтраховойСлучай (Больничный травма на производстве новый страховой случай)
- ОтпускПоБеременностиИРодамПоУходуЗаРебенкомНовыйСтраховойСлучай (Отпуск по беременности и родам, по уходу за ребенком, новый страховой случай)
- БольничныйПрочийНовыйСтраховойСлучай (Больничный прочий новый страховой случай)
- БольничныйОбщееЗаболеваниеНовыйСтраховойСлучай (Больничный общее заболевание новый страховой случай)
- БольничныйЗаСчетРаботодателя (Больничный за счет работодателя)
- БольничныйЗаСчетРаботодателяНовыйСтраховойСлучай (Больничный за счет работодателя новый страховой случай)
