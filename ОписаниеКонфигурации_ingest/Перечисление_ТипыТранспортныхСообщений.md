# Перечисление: ТипыТранспортныхСообщений (Типы транспортных сообщений)

## Значения

- ПервичноеСообщениеСодержащееОтчетность (Первичное сообщение, содержащее файл отчетности)
- ПодтверждениеОтправкиОтчетности (Подтверждение отправки файла отчетности)
- ПереподписанноеПодтверждениеОтправкиОтчетности (Переподписанное налогоплательщиком подтверждение отправки файла отчетности)
- КвитанцияОПриемеФайлаОтчетности (Квитанция о приеме файла отчетности налоговым органом)
- ПротоколВходногоКонтроляОтчетности (Протокол входного контроля файла отчетности)
- ПереподписанныйПротоколВходногоКонтроляОтчетности (Переподписанный протокол входного контроля отчетности)
- УведомлениеОбУточненииНБО (Уведомление об уточнении НБО)
- Протокол2НДФЛ (Протокол приема 2- НДФЛ)
- НеформализованныйДокументНО (Документ налогового органа)
- ПодтверждениеНаНеформализованныйДокументНО (Подтверждение приема документа налогового органа)
- НеформализованныйДокументНалогоплательщика (Документ налогоплательщика)
- ПодтверждениеОтправкиНеформализованногоДокументаНалогоплательщика (Подтверждение отправки документа налогоплательщика)
- ПереподписанноеПодтверждениеОтправкиНеформализованногоДокументаНалогоплательщика (Переподписанное налогоплательщиком подтверждение отправки документа налогоплательщика)
- ПодтверждениеДоставкиНеформализованногоДокументаНалогоплательщика (Квитанция о доставке документа налогоплательщика в налоговый орган)
- ЗапросИОН (Запрос на информационное обслуживание налогоплательщика)
- ПодтверждениеОтправкиЗапросаИОН (Подтверждение отправки запроса на информационное обслуживание налогоплательщика)
- ПереподписанноеПодтверждениеОтправкиЗапросаИОН (Переподписанное налогоплательщиком подтверждение отправки запроса на информационное обслуживание)
- КвитанцияОПриемеЗапросаИОН (Квитанция о приеме запроса на информационное обслуживание налоговым органом)
- ПротоколВходногоКонтроляЗапросаИОН (Протокол входного контроля запроса на информационное обслуживание)
- ПереподписанныйПротоколВходногоКонтроляЗапросаИОН (Переподписанный налогоплательщиком протокол входного контроля запроса на информационное обслуживание)
- ОтветНаЗапросИОН (Ответ налогового органа на запрос на информационное обслуживание)
- ПереподписанныйОтветНаЗапросИОН (Переподписанный налогоплательщиком ответ на запрос на информационное обслуживание)
- КритическаяОшибка (Критическая ошибка)
- Прочее (Прочее)
- ПервичноеСообщениеСодержащееОтчетностьПФР (Первичное сообщение, содержащее файл отчетности для ПФР)
- ПодтверждениеПолученияОтчетностиПФР (Подтверждение получения отчетности)
- ПротоколПФР (Протокол)
- ПротоколКвитанцияПФР (Квитанция на протокол)
- УведомлениеОбОшибкеПФР (Уведомление об ошибке при обмене с ПФР)
- ПисьмоВходящееПФР (Входящее письмо)
- ПисьмоВходящееКвитанцияПФР (Квитанция на входящее письмо)
- ПисьмоИсходящееПФР (Исходящее письмо)
- ПисьмоИсходящееКвитанцияПФР (Квитанция на исходящее письмо)
- ДекларацияНП (Первичное сообщение, содержащее отчетность)
- ПодтверждениеДекларацияНО (Подтверждение даты отправки)
- ИзвещениеПодтверждениеНП (Извещение о получении подтверждения даты отправки)
- ИзвещениеДекларацияНО (Извещение о получении декларации)
- РезультатПриемаДекларацияНО (Результат приема декларации)
- ИзвещениеРезультатПриемаНП (Извещение о получении результата приема)
- РезультатОбработкиДекларацияНО (Результат обработки декларации)
- ИзвещениеРезультатОбработкиНП (Извещение о получении результата обработки)
- УведомлениеОбОшибкеФНС (Уведомление об ошибке при обмене с ФНС)
- Форма2НДФЛНП (Форма 2-НДФЛ)
- ПодтверждениеФорма2НДФЛНО (Подтверждение даты отправки 2-НДФЛ)
- ИзвещениеФорма2НДФЛНО (Извещение о получении формы 2-НДФЛ)
- РезультатПриемаФорма2НДФЛНО (Результат приема формы 2-НДФЛ)
- ОбращениеНП (Обращение налогоплательщика)
- ИзвещениеОбращениеНО (Извещение о получении обращения налогоплательщика)
- РезультатПриемаОбращениеНО (Результат приема обращения налогоплательщика)
- ПодтверждениеОбращениеНО (Подтверждение даты отправки обращения налогоплательщика)
- ПисьмоНО (Письмо налогового органа)
- РассылкаНО (Рассылка налогового органа)
- ИзвещениеПисьмоНП (Извещение о получении письма налогового органа)
- ИзвещениеРассылкаНП (Извещение о получении рассылки налогового органа)
- ЗапросНП (Запрос на информационное обслуживание налогоплательщика)
- ПодтверждениеЗапросНО (Подтверждение даты отправки запроса налогоплательщика)
- ИзвещениеЗапросНО (Извещение о получении запроса налогоплательщика)
- РезультатПриемаЗапросНО (Результат приема запрос налогоплательщика)
- РезультатОбработкиЗапросНО (Результат обработки запрос налогоплательщика)
- ПервичноеСообщениеСодержащееОтчетностьФСГС (Первичное сообщение, содержащее файл отчетности)
- ПодтверждениеДатыОтправкиФСГС (Подтверждение даты отправки документа)
- ИзвещениеПодтверждениеДатыОтправкиФСГС (Извещение о получении респондентом подтверждения даты отправки документа)
- ПереподписанноеПодтверждениеОтправкиОтчетностиФСГС (Переподписанное подтверждение отправки файла отчетности)
- ИзвещениеОПолученииОтчетностиФСГС (Извещение о получении файла отчетности)
- ПротоколВходногоКонтроляОтчетностиФСГС (Протокол входного контроля отчетности)
- ИзвещениеПротоколВходногоКонтроляОтчетностиФСГС (Извещение о получении респондентом протокола входного контроля отчетности)
- ПисьмоВходящееФСГС (Входящее письмо)
- ИзвещениеПисьмоВходящееФСГС (Извещение о получении письма респондентом)
- ПисьмоИсходящееФСГС (Исходящее письмо)
- ИзвещениеПисьмоИсходящееФСГС (Извещение о получении письма органом ФСГС)

- РезультатОбработкиЗаявлениеРФНО (Результат обработки заявления)
- ИзвещениеРезультатОбработкиРФНП (Извещение о получении результата обработки заявления)

- ДокументНО (Документ налогового органа)
- ИзвещениеДокументНП (Извещение о получении документа налогоплательщиком)
- РезультатПриемаДокументНП (Результат приема документа налогоплательщиком)

- ОтветНаЗапросПФР (Ответ на запрос ПФР)
