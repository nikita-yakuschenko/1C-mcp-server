# Перечисление: ТООшибкиФР (ТООшибки ФР)

## Значения

- ОшибкаНеобходимПарольАдминистратора (Для выполнения данной операции необходим корректный пароль администратора)
- ОшибкаНеобходимПарольКассира (Для выполнения данной операции необходим корректный пароль кассира)
- ОшибкаНетВозможностиПечатиПКО (Данный фискальный регистратор не имеет возможности отражать в чеке факт наличных расчетов с юридическими лицами)
- ОшибкаНетНаличных (Недостаточно наличных средств в кассе для выполнения операции)
- ОшибкаРежимПечатиЧека (Выполнение данного действия невозможно в режиме печати чека)
- ОшибкаСуммыНедостаточно (Переданной суммы недостаточно для оплаты)
