# Перечисление: КодыОперацийВыпускПродукции (Коды операций выпуск продукции)

## Значения

- ВозвратМатериаловИзПроизводстваРасч (Возврат материалов из производства по расчетной стоимости)
- ВыпускПродукцииПоФиксированнойСтоимости (Выпуск продукции по фиксированной стоимости)
- ВыпускПродукцииФиксВнешниеУслуги (Выпуск продукции фикс (внешние услуги))
- ВыпускПродукцииФиксВнутрУслугиНаПостЗатраты (Выпуск продукции фикс (внутр. услуги на постоянные затраты))
- ВыпускПродукцииФиксВнутрУслугиНаПроизводство (Выпуск продукции фикс (внутр. услуги на производство))
- ВыпускПродукцииФиксВнутрУслугиНаСчет (Выпуск продукции фикс (внутр. услуги на счета бух.учета))
- ВыпускПродукцииФиксНаСклад (Выпуск продукции фикс. (на склад))
- СписаниеНЗП (Списание НЗП)
- ФормированиеФактСебестоимости (Формирование фактической себестоимости)
- ОприходованиеНЗП (Оприходование НЗП)
- КорректировкаНЗПРасч (Корректировка незавершенного производства по расчетной стоимости)
- ВозвратМатериаловИзПроизводстваФикс (Возврат материалов из производства по фиксированной стоимости)
- СписаниеНЗПФикс (Списание НЗП по фиксированной стоимости)
- КорректировкаНЗПФикс (Корректировка незавершенного производства по фиксированной стоимости)
