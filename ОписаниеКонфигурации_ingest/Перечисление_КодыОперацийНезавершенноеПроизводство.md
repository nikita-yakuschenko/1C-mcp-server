# Перечисление: КодыОперацийНезавершенноеПроизводство (Коды операций незавершенное производство)

## Значения

- СписаниеПартийВПроизводствоОперативно (Списание партий в производство оперативно)
- СписаниеКоличестваВПроизводствоОперативно (Списание количества в производство оперативно)
- СписаниеПрочихЗатратВПроизводство (Списание прочих затрат в производство)
- ВозвратМатериаловИзПроизводстваФикс (Возврат материалов из производства по фиксированной стоимости)
- ВнутренниеУслугиНаПроизводство (Внутренние услуги на производство)
- УслугиКонтрагентаНаПроизводство (Услуги контрагента на производство)
- ИнвентаризацияНЗП (Инвентаризация незавершенного производства)
- СписаниеНЗПФикс (Списание незавершенного производства (по фиксированной стоимости))
- СписаниеНЗПРегламент (Списание незавершенного производства регламент.)
- СписаниеОХР_Регламент (Списание ОХР (26-й счет) (регламент.))
- СписаниеОПР_Регламент (Списание ОПР (25-й счет) (регламент.))
- СписаниеОтрицЗатратРегламент (Списание отрицательных затрат (регламент.))
- СписаниеБракВПроизводстве_Регламент (Списание затрат по браку в пр-ве (28-й счет) (регламент.))
- ОтрицательныеЗатраты (Отрицательные затраты)
- ОприходованиеНЗП (Оприходование НЗП)
- ОприходованиеНЗПФикс (Оприходование НЗП (фикс. стоимость))
- КорректировкаНЗПФикс (Корректировка незавершенного производства  (фикс. стоимость))
- РаспределениеНЗПРегламент (Распределение незавершенного производства (регламент.))
