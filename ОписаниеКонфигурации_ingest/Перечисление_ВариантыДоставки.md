# Перечисление: ВариантыДоставки (Варианты доставки)

## Значения

- ДоставкаПоставщикомНаСклад (Доставка поставщиком на склад)
- ДоставкаПоставщикомНаСкладПодЗаказы (Доставка поставщиком на склад под заказы)
- ДоставкаПоставщикомНаСкладПодЗаказыИНаСклад (Доставка поставщиком на склад под заказы и без заказов)
- ДоставкаПоставщикомПоТочкам (Доставка поставщиком по точкам)
- ДоставкаПеревозчикомНаСклад (Доставка перевозчиком на склад)
- ДоставкаПеревозчикомПоТочкам (Доставка перевозчиком по точкам)
- ДоставкаПеревозчикомПоТочкамИНаСклад (Доставка перевозчиком по точкам и остаток на склад)
