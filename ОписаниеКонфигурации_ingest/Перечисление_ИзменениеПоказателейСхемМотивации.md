# Перечисление: ИзменениеПоказателейСхемМотивации (Изменение показателей схем мотивации)

## Значения

- НеИзменяется (Вводится только при кадровых изменениях)
- ИзменяетсяПриРасчете (Может изменяться при расчете)
- ВводитсяПриРасчете (Вводится только при расчете)
- Ежемесячно (Ежемесячно перед расчётом зарплаты)
- Периодически (Периодически)
