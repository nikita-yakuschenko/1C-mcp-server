# Перечисление: ОтношениеКВоинскомуУчету (Отношение к воинскому учету)

## Значения

- Состоит (Состоит на воинском учете)
- Встает (Встает на воинский учет)
- НеСостоит (Не состоит на воинском учете (но должен))
- СнятПоВозрасту (Снят с воинского учета по возрасту)
- СнятПоСостояниюЗдоровья (Снят с воинского учета по состоянию здоровья)
