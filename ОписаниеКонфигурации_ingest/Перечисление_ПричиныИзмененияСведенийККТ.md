# Перечисление: ПричиныИзмененияСведенийККТ (Причины изменения сведений о ККТ)

## Значения

- ЗаменаФН (Замена фискального накопителя)
- ЗаменаОФД (Замена оператора фискальных данных)
- ИзменениеНаименованияПользователя (Изменение наименования пользователя контрольно-кассовой техники)
- ИзменениеАдреса (Изменение адреса и (или) места установки (применения) контрольно-кассовой техники)
- ПереводВРежимПередачиДанных (Перевод ККТ из автономного режима в режим передачи данных)
- ПереводВАвтономныйРежим (Перевод ККТ из режима передачи данных в автономный режим)
- ИзменениеВерсии (Изменение версии модели ККТ)
- ИзменениеСистемНалогообложения (Изменение перечня систем налогообложения, применяемых при осуществлении расчетов)
- ИзменениеНомераУстройства (Изменение номера автоматического устройства для расчетов, в составе которого применяется ККТ)
- ПереводВНеавтоматическийРежим (Перевод ККТ из автоматического режима в неавтоматический режим (осуществление расчетов кассиром))
- ПереводВАвтоматическийРежим (Перевод ККТ из неавтоматического режима (осуществление расчетов кассиром) в автоматический режим)
- ВключениеБСО (Перевод ККТ из режима, не позволяющего формировать БСО, в режим, позволяющий формировать БСО)
- ВыключениеБСО (Перевод ККТ из режима, позволяющего формировать БСО, в режим, не позволяющий формировать БСО)
- ВключениеРежимаРасчетовВИнтернет (Перевод ККТ в режим расчетов в сети Интернет)
- ВыключениеРежимаРасчетовВИнтернет (Перевод ККТ из режима расчетов в сети Интернет)
- ВключениеРежимаАгента (Перевод ККТ в режим, позволяющий оказывать услуги платежного или банковского агента)
- ВыключениеРежимаАгента (Перевод ККТ в режим не позволяющий оказывать услуги платежного или банковского агента)
- ВключениеРежимаАзартныхИгр (Перевод ККТ в режим, позволяющий применять ККТ при приеме ставок)
- ВыключениеРежимаАзартныхИгр (Перевод ККТ в режим, не позволяющий применять ККТ при приеме ставок)
- ВключениеРежимаЛотереи (Перевод ККТ в режим, позволяющий применять ККТ при реализации лотерейных билетов)
- ВыключениеРежимаЛотереи (Перевод ККТ в режим, не позволяющий применять ККТ при реализации лотерейных билетов)
- ИзменениеВерсииФФД (Изменение версии ФФД)
- Иное (Иные причины)
