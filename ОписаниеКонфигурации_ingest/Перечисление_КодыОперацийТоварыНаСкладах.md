# Перечисление: КодыОперацийТоварыНаСкладах (Коды операций товары на складах)

## Значения

- СписаниеПартийВПроизводствоОперативно (Списание партий в производство оперативно)
- ВыпускПродукцииФиксНаСклад (Выпуск продукции фикс. (на склад))
- ВозвратМатериаловИзПроизводстваФикс (Возврат материалов из производства по фиксированной стоимости)
- ВозвратМатериаловИзПроизводстваРасч (Возврат материалов из производства по расчетной стоимости)
- ВозвратПродукцииИзПроизводстваФикс (Возврат продукции из производства по фиксированной стоимости)
- ВыпускПоОперацииСтоимость (Выпуск по операции стоимость)
- ВозвратМатериаловИзЭксплуатации (Возврат материалов из эксплуатации)
