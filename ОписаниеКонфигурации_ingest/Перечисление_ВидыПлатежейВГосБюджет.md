# Перечисление: ВидыПлатежейВГосБюджет (Виды платежей в бюджет (фонды))

## Значения

- Налог (Налог (взносы): начислено / уплачено)
- НалогСам (Налог (взносы): доначислено / уплачено (самостоятельно))
- НалогАкт (Налог (взносы): доначислено / уплачено (по акту проверки))
- Штраф (Штраф: начислено / уплачено)
- РасходыПоСтрахованию (Расходы по страхованию)
- ПениСам (Пени: доначислено / уплачено (самостоятельно))
- ПениАкт (Пени: начислено / уплачено (по акту проверки))
- ПолученоИзФонда (Получено от исполнительного органа ФСС РФ)
- НеПринято (Не принято к зачету)
- ВзносыБезСпецоценки (Взносы по доп.тарифам при отсутствии спецоценки)
