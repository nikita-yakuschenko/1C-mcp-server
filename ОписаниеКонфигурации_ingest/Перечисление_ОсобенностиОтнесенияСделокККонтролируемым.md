# Перечисление: ОсобенностиОтнесенияСделокККонтролируемым (Особенности отнесения сделок к контролируемым)

## Значения

- ВзаимозависимыеЛица (Взаимозависимые лица)
- ВзаимозависимыеЛицаПлательщикиНДПИ (Одно из взаимозависимых лиц является плательщиком НДПИ)
- ВзаимозависимыеЛицаНаСпецрежимах (Одно из взаимозависимых лиц является плательщиком ЕНВД или ЕСХН)
- ВзаимозависимыеЛицаПлательщикиНалогаНаПрибыль (Одно из взаимозависимых лиц является плательщиком налога на прибыль)
- ВзаимозависимыеЛицаОсобаяЭкономическаяЗона (Одно из взаимозависимых лиц является резидентом особой экономической зоны)
- ВзаимозависимыеЛицаДеятельностьМорскогоМесторождения (Одно из взаимозависимых лиц является оператором морского месторождения)
- ВзаимозависимыеЛицаУчастникиРегиональногоИнвестиционногоПроекта (Одно из взаимозависимых лиц является участником регионального инвестиционного проекта)
- ВзаимозависимыеЛицаИнвестиционныйВычет (Одна из сторон сделки применяет инвестиционный налоговый вычет по налогу на прибыль)
- МинимальнаяСуммаДляВключенияВУведомление (Минимальная сумма для включения в уведомление)
- НезависимыеИностранныеЛица (Независимые иностранные лица)
- ВзаимозависимыеЛицаОсвобождениеНДС (Одно из взаимозависимых лиц освобождено от уплаты НДС (Сколково))
- ВзаимозависимыеЛицаПрочие (Взаимозависимые лица прочие)
