# Перечисление: РежимыВыгрузкиОбъектовОбмена (Режимы выгрузки объектов обмена)

## Значения

- ВыгружатьВсегда (Выгружать всегда)
- ВыгружатьПоУсловию (Выгружать по условию)
- ВыгружатьПриНеобходимости (Выгружать при необходимости)
- ВыгружатьВручную (Выгружать вручную)
- НеВыгружать (Не выгружать)
