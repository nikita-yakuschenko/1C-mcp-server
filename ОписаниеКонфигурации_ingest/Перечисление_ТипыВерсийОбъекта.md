# Перечисление: ТипыВерсийОбъекта (Типы версий объекта)

## Значения

- ИзмененоПользователем (Изменено пользователем)
- ПринятыеДанныеПоКоллизии (Принятые данные по коллизии)
- НепринятыеДанныеПоКоллизии (Непринятые данные по коллизии)
- НепринятыеДанныеПоДатеЗапретаОбъектСуществует (Непринятый по дате запрета объект существует в базе)
- НепринятыеДанныеПоДатеЗапретаОбъектНеСуществует (Непринятый по дате запрета объект не существует в базе)
